# HiShare

## Overview

HiShare is a patient-owned health data platform that enables individuals to securely manage their health information, control who can access it, and participate in medical research opportunities in exchange for compensation.

The platform is built around the principle that patients should own their healthcare data and decide how, when, and with whom it is shared.

## Features

- Patient dashboard with health profile, permissions, records, research opportunities, and activity timeline.
- Health records module for medical history, prescriptions, lab results, imaging, vaccinations, and allergies.
- Research marketplace with study listings, eligibility status, and compensation details.
- Rewards center to track earnings, participation status, and payment history.
- Reporting experience with data and consent analytics.
- Authentication experience with sign-in and sign-up workflows.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173
